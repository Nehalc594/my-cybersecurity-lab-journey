# Room: Defensive Security Intro

**Section:** Start Your Cyber Security Journey  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Basics of defensive security
- Monitoring and protecting systems
- Incident detection and response concepts

## Key Concepts
- Defense focuses on prevention, detection, and response
- Logs and alerts are critical
- Defense is continuous

## Security Perspective
- Offensive: attempts to bypass defenses
- Defensive: detects and mitigates attacks

## Key Takeaways
- Detection is as important as prevention
- Visibility is critical
- Defense requires constant improvement
