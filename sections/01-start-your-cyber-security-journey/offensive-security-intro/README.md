# Room: Offensive Security Intro

**Section:** Start Your Cyber Security Journey  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Introduction to offensive security
- Role of ethical hackers
- Difference between offense and defense

## Key Concepts
- Offensive security identifies weaknesses before attackers do
- Authorization and scope are mandatory
- Findings improve defensive posture

## Security Perspective
- Offensive: simulate real attack techniques
- Defensive: use findings to harden systems

## Key Takeaways
- Ethical hacking is permission-based
- Tools matter less than methodology
- Security is both offensive and defensive
