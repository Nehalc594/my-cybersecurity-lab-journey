# Room: Search Skills

**Section:** Start Your Cyber Security Journey  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Importance of effective search techniques in cybersecurity
- Using search engines to find technical information
- Finding documentation, exploits, and security references

## Key Concepts
- Strong search skills are critical for security professionals
- Knowing *what* to search is as important as *how* to search
- Reliable sources matter more than random results

## Security Perspective
- Offensive: researching vulnerabilities, tools, and techniques
- Defensive: finding mitigations, patches, and best practices

## Key Takeaways
- Search engines are daily tools for security professionals
- Official documentation and trusted sources are preferred
- Continuous learning depends on strong research skills
