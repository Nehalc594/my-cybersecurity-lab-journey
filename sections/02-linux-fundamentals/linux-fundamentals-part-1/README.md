**Section:** Linux Fundamentals  
**Difficulty:** Beginner

## What This Room Covers
- Purpose of Linux in cybersecurity
- Basic Linux system structure
- Introduction to terminal usage

## Key Concepts
- Linux is widely used in servers and security tools
- Most security tooling is Linux-based
- The terminal provides powerful control over the system

## Security Perspective
- Offensive: attackers often rely on Linux-based tooling
- Defensive: servers and security infrastructure commonly run on Linux

## Key Takeaways
- Comfort with Linux is essential in cybersecurity
- Understanding the file system is foundational
- Command-line interaction is a core skill
