**Section:** Linux Fundamentals  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Linux processes and services
- Managing running applications
- Basic system monitoring concepts

## Key Concepts
- Processes represent running programs
- Services often run in the background
- System monitoring helps detect abnormal behavior

## Security Perspective
- Offensive: attackers may hide malicious processes or abuse services
- Defensive: monitoring processes helps identify compromise

## Key Takeaways
- Understanding processes is essential for system visibility
- Long-running services are common attack targets
- Monitoring is a core defensive security practice
