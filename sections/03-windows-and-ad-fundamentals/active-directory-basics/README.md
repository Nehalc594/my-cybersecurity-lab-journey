# Room: Active Directory Basics

**Section:** Windows and AD Fundamentals  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Purpose of Active Directory in enterprise environments
- Centralized identity and access management
- Core AD components and concepts

## Key Concepts
- Active Directory manages users, computers, and resources centrally
- Authentication and authorization are core AD functions
- AD misconfigurations can have enterprise-wide impact

## Security Perspective
- Offensive: attackers target AD to gain broad access across networks
- Defensive: securing AD is critical to protecting the organization

## Key Takeaways
- Active Directory is the backbone of most enterprise environments
- Compromise of AD often means compromise of the entire network
- Strong identity and access controls are essential for security
