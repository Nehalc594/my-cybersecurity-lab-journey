# Room: Windows Fundamentals 1

**Section:** Windows and AD Fundamentals  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Overview of the Windows operating system
- Purpose of core Windows components
- Basic navigation and system structure

## Key Concepts
- Windows is widely used in enterprise environments
- The operating system manages users, processes, and resources
- Understanding Windows internals is essential for security analysis

## Security Perspective
- Offensive: attackers target common Windows components and misconfigurations
- Defensive: defenders rely on Windows features for access control and logging

## Key Takeaways
- Windows is a primary target due to its widespread use
- Familiarity with Windows internals improves threat detection
- Foundational knowledge is required before deeper AD and security topics
