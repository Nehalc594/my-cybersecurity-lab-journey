# Room: Windows Fundamentals 2

**Section:** Windows and AD Fundamentals  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Windows user accounts and groups
- File system permissions and access control
- Basic system management concepts

## Key Concepts
- Windows uses users and groups to control access
- Permissions determine who can access files and resources
- Misconfigured permissions can lead to security risks

## Security Perspective
- Offensive: attackers abuse weak permissions to gain or expand access
- Defensive: proper user and permission management limits impact of compromise

## Key Takeaways
- Least privilege is critical in Windows environments
- User and group management is a core security responsibility
- Access control misconfigurations are common attack paths
