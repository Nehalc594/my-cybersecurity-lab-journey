# Room: Windows Fundamentals 3

**Section:** Windows and AD Fundamentals  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Windows services and processes
- System monitoring and logging concepts
- Introduction to Windows security mechanisms

## Key Concepts
- Services often run with elevated privileges
- Logs provide visibility into system activity
- Monitoring is essential for detecting malicious behavior

## Security Perspective
- Offensive: attackers may abuse services or hide malicious activity
- Defensive: logs and monitoring help detect and investigate incidents

## Key Takeaways
- Visibility into system activity is critical for security
- Services are common targets due to privilege levels
- Logging is foundational for incident response
