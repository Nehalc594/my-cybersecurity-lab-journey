# Room: Networking Concepts

**Section:** Networking  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- What computer networks are and why they exist
- How devices communicate over networks
- Basic terminology used in networking

## Key Concepts
- Networks connect devices to share data and resources
- Communication follows defined rules (protocols)
- Understanding networking is essential for identifying attack paths

## Security Perspective
- Offensive: attackers exploit network exposure and misconfigurations
- Defensive: defenders design networks to limit access and visibility

## Key Takeaways
- Networking knowledge underpins most security work
- Poor network design increases attack surface
- Visibility and segmentation improve security posture
