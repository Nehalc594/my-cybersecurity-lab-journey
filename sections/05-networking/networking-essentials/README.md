# Room: Networking Essentials

**Section:** Networking  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Core components that make up a network
- Role of IP addresses, ports, and devices
- How data moves between systems

## Key Concepts
- IP addresses identify devices on a network
- Ports allow multiple services to run on a single system
- Network devices control traffic flow and connectivity

## Security Perspective
- Offensive: attackers identify exposed services through IPs and ports
- Defensive: defenders manage exposure using segmentation and filtering

## Key Takeaways
- Understanding IPs and ports is essential for enumeration
- Every open service increases attack surface
- Network design directly impacts security
