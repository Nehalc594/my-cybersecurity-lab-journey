## Key Concepts
- Tcpdump captures packets directly from network interfaces

## Security Perspective
- Offensive: attackers may capture traffic to understand network behavior
- Defensive: defenders use tcpdump for rapid investigation and incident response

## Key Takeaways
- Command-line packet capture is powerful and efficient
- Tcpdump is commonly used on servers and headless systems
- Traffic analysis supports troubleshooting and threat detection
