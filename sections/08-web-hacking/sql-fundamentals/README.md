# Room: SQL Fundamentals

**Section:** Web Hacking  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Purpose of databases in web applications
- Basic structure of SQL-based databases
- How applications interact with stored data

## Key Concepts
- Databases store and manage application data
- SQL is used to query and manipulate data
- Improper query handling can expose sensitive information

## Security Perspective
- Offensive: attackers target weak query handling to access data
- Defensive: parameterized queries and validation reduce risk

## Key Takeaways
- Databases are high-value attack targets
- Input handling directly affects database security
- Secure design prevents many data-related vulnerabilities
