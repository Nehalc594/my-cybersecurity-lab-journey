# Room: Shells Overview

**Section:** Offensive Security Tooling  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- What shells are in cybersecurity contexts
- Difference between interactive and non-interactive shells
- Why shells represent significant system access

## Key Concepts
- A shell allows command execution on a system
- Shell access often follows successful exploitation
- Shells can be used for administration or abuse

## Security Perspective
- Offensive: testers assess impact after gaining shell access
- Defensive: defenders monitor for unexpected shell activity

## Key Takeaways
- Shell access indicates a serious security breach
- Monitoring execution behavior is critical
- Limiting shell access reduces attack impact
