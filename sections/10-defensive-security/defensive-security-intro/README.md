# Room: Defensive Security Intro

**Section:** Defensive Security  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Purpose of defensive security in organizations
- Role of blue teams in protecting systems
- Core defensive security objectives

## Key Concepts
- Defensive security focuses on prevention, detection, and response
- Visibility into systems and networks is essential
- Defense is a continuous process, not a one-time task

## Security Perspective
- Offensive: attempts to bypass controls
- Defensive: designs, monitors, and improves controls

## Key Takeaways
- Strong defense reduces attack impact
- Detection is as important as prevention
- Continuous monitoring improves resilience
