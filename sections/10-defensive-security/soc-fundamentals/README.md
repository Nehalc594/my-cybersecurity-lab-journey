# Room: SOC Fundamentals

**Section:** Defensive Security  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Purpose of a Security Operations Center (SOC)
- Core SOC responsibilities and workflows
- How alerts are monitored and handled

## Key Concepts
- SOC teams monitor, detect, and respond to security events
- Alerts must be triaged and prioritized
- Clear processes reduce response time and errors

## Security Perspective
- Offensive: attacker activity generates detectable signals
- Defensive: SOC analysts investigate and respond to those signals

## Key Takeaways
- SOCs are the frontline of defense
- Effective triage prevents alert fatigue
- Process and visibility are critical to success
