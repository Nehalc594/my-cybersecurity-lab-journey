# Room: IDS Fundamentals

**Section:** Security Solutions  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Purpose of intrusion detection systems (IDS)
- Difference between detection and prevention
- Role of IDS in monitoring network and system activity

## Key Concepts
- IDS monitors traffic and behavior for suspicious activity
- Alerts are generated when potential threats are detected
- Detection focuses on visibility, not blocking

## Security Perspective
- Offensive: attacker activity may trigger detection alerts
- Defensive: defenders analyze IDS alerts to identify threats

## Key Takeaways
- Detection complements preventive controls
- IDS improves visibility into malicious activity
- Alert quality affects investigation effectiveness
