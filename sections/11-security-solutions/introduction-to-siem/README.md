# Room: Introduction to SIEM

**Section:** Security Solutions  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Purpose of Security Information and Event Management (SIEM)
- How logs are centralized and correlated
- Role of SIEM in detection and investigations

## Key Concepts
- SIEM aggregates logs from multiple sources
- Correlation provides context across events
- Alerts help identify suspicious activity

## Security Perspective
- Offensive: attacker activity generates correlated signals
- Defensive: analysts use SIEM to detect and investigate threats

## Key Takeaways
- Centralized visibility improves detection
- Context reduces false positives
- SIEM is a core SOC technology
