## Key Concepts
- Capability analysis focuses on behavior, not signatures
- Static analysis can reveal malicious functionality

## Security Perspective
- Offensive: malware is designed with specific capabilities
- Defensive: analysts identify capabilities to assess risk and impact

## Key Takeaways
- Capability-based analysis speeds investigations
- Knowing what malware can do informs response
- Static analysis reduces execution risk
