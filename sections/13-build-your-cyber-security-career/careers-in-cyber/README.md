# Room: Careers in Cyber

**Section:** Build Your Cyber Security Career  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Overview of different cybersecurity roles
- How skills map to career paths
- Expectations for entry-level and advanced roles

## Key Concepts
- Cybersecurity includes offensive, defensive, and governance roles
- Career paths are often non-linear
- Continuous learning is required in security careers

## Security Perspective
- Offensive: red team and testing roles focus on finding weaknesses
- Defensive: blue team roles focus on detection, response, and prevention

## Key Takeaways
- There is no single “correct” path into cybersecurity
- Practical skills and fundamentals matter
- Growth requires persistence and adaptability
