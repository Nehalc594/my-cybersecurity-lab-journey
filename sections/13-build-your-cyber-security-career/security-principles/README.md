# Room: Security Principles

**Section:** Build Your Cyber Security Career  
**Difficulty:** Beginner  
**Status:** Completed  

---

## What This Room Covers
- Core principles that guide cybersecurity decisions
- Why security design must balance risk and usability
- Importance of consistent security thinking

## Key Concepts
- Security is about risk management, not absolute prevention
- Defense-in-depth improves resilience
- Least privilege limits the impact of compromise

## Security Perspective
- Offensive: attackers exploit gaps where principles are ignored
- Defensive: applying principles consistently reduces systemic risk

## Key Takeaways
- Strong principles guide better security decisions
- No single control is sufficient on its own
- Security must align with business objectives
