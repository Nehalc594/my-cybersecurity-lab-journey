## What This Room Covers
- Identity, Authentication, Authorization, and Accounting failures
- How access control weaknesses lead to compromise
- Why identity is a primary attack target

## Key Concepts
- Authentication verifies who a user is
- Authorization defines what a user can access
- Weak identity controls enable privilege abuse

## Security Perspective
- Offensive: attackers exploit weak authentication and access controls
- Defensive: strong identity governance reduces breach impact

## Key Takeaways
- Identity is the new security perimeter
- Least privilege limits damage
- Strong authentication and logging are critical
