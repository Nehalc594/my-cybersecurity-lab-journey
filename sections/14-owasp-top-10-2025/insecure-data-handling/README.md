## What This Room Covers
- Risks related to improper handling of sensitive data
- How data exposure occurs in applications
- Importance of secure storage and transmission

## Key Concepts
- Sensitive data must be protected at rest and in transit
- Improper storage or transmission leads to data breaches
- Data classification helps apply appropriate protections

## Security Perspective
- Offensive: attackers target exposed or weakly protected data
- Defensive: encryption and access controls protect sensitive information

## Key Takeaways
- Data protection is a core security responsibility
- Encryption reduces impact of compromise
- Secure data handling prevents regulatory and reputational damage
